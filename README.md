# CodeAlpha_Hangman_Game

 # Hangman Game 🎯


A simple text-based Hangman game built with Python.

The player guesses a hidden word one letter at a time before the stick figure is fully drawn.

## How to Play

A random word is selected and displayed as underscores: _ _ _ _ _ _

Type a single letter and press Enter.

A correct guess reveals the letter in the word.

A wrong guess adds a body part to the gallows.

Win by guessing all letters before 6 wrong guesses.
